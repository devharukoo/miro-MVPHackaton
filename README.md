# 🎨 UnitBoard — Miro Clone MVP

Прототип бесконечной интерактивной доски для совместной работы, созданный в рамках хакатона. Проект позволяет создавать задачи, перемещать их и менять стили в реальном времени.

---

## 🚀 Основные фичи
*   **Бесконечный холст:** Свободное перемещение по рабочей области с помощью мыши и навигация (зум/панорамирование).
*   **Real-time синхронизация:** Все изменения (создание, движение, ресайз карточек) мгновенно отображаются у других пользователей через Socket.io.
*   **Управление задачами:** Создание двойным кликом, изменение текста прямо в карточке и удаление через клавишу Delete.
*   **Кастомизация:** Смена названия доски и цветовой темы интерфейса «на лету» через панель настроек.
*   **Persistence:** Все данные сохраняются в базе данных MongoDB.

---

## 🛠 Стек технологий
*   **Frontend:** HTML5 (Canvas), CSS3, Vanilla JavaScript.
*   **Backend:** Node.js, Express.
*   **Real-time:** Socket.io.
*   **Database:** MongoDB + Mongoose.

## 📦 Как запустить проект локально

Чтобы запустить проект у себя на компьютере, выполните следующие шаги:

### 1. Клонирование репозитория
```bash
git clone [https://github.com/devharukoo/miro-MVPHackaton.git](https://github.com/devharukoo/miro-MVPHackaton.git)
cd miro-MVPHackaton

2. Установка зависимостей
Убедитесь, что у вас установлен Node.js.

Bash
npm install

3. Настройка окружения
Создайте файл .env в корневой папке и добавьте туда строку подключения к вашей MongoDB:

MONGODB_URI=your_mongodb_connection_string
PORT=3000
4. Запуск сервера
Bash
node app.js
После этого откройте в браузере: http://localhost:3000
